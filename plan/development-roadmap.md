# Nexus 外脑思维链路管理系统 — 开发路线图

## 愿景

基于 COP 模型（Capture → Orchestrate → Produce）构建一个**认知伙伴**，让思考可见、可塑、可延伸。

## 当前状态（已完成）

- [x] Phase 1: 前端模块化（Vite + TypeScript，24 个模块）
- [x] Phase 2: 后端 LLM 集成（FastAPI + Anthropic SDK，4 个 AI 端点）
- [x] Phase 3: 持久化 + 多空间（SQLite + 自动保存 + 空间切换）
- [x] 核心画布：无限画布、卡片拖拽、连线、5 种关系标签
- [x] 版本管理：Git 式分支、快照、回溯、版本图
- [x] AI 质询：苏格拉底式质疑、思维流分析
- [x] 结论固化：卡片群 → 摘要卡片

## 架构演进路径

```
当前：Vite 前端 + FastAPI Python 后端
  ↓
Phase 4-6：继续 Web 架构，打磨核心逻辑
  ↓
Phase 7+：Tauri 包装（Rust 后端 + 现有 Web 前端）
```

---

## Phase 4：编排层增强（Orchestrate++）

> 目标：让画布从"连线工具"进化为"思维编排引擎"

### 4.1 连线体验优化
- [ ] 贝塞尔曲线连线（替代直线，视觉更自然）
- [ ] 连接端口智能吸附（拖拽时高亮最近端口）
- [ ] 连接线颜色按关系类型区分
- [ ] 连接线粗细反映关联强度（可选）
- **预估**：2-3 天

### 4.2 卡片分组与聚类
- [ ] 框选多张卡片 → 右键"创建分组"
- [ ] 分组有背景色块 + 标题
- [ ] 分组可折叠/展开
- [ ] 分组内卡片相对位置固定
- **预估**：3-4 天

### 4.3 大纲视图
- [ ] 画布内容一键转大纲（按连接关系推导层级）
- [ ] 大纲中可拖拽调整层级
- [ ] 大纲修改同步回画布
- [ ] 视图切换动画
- **预估**：4-5 天

### 4.4 双链引用 `[[ ]]`
- [ ] 卡片文本中输入 `[[` 触发卡片搜索
- [ ] 选择后自动创建引用关系
- [ ] 被引用卡片显示反向链接计数
- [ ] 双链密度达到阈值时提示"升级为专题"
- **预估**：5-6 天

### 4.5 自定义关系类型
- [ ] 关系标签管理面板（新增/编辑/删除）
- [ ] 每种关系可设置颜色、图标
- [ ] 关系模板导入/导出
- **预估**：2-3 天

**Phase 4 总预估**：16-21 天

## Phase 5：AI 引擎主动化（AI++）

> 目标：AI 从"被动回答"进化为"主动伙伴"

### 5.1 关系发现者
- [ ] 新增卡片时，自动扫描现有卡片库
- [ ] 基于语义相似度推荐潜在关联（embedding 向量）
- [ ] 推荐结果以浮窗展示，一键确认/忽略
- [ ] 后端新增 `/api/llm/similar` 端点
- **预估**：5-6 天（需引入 embedding 模型或本地向量搜索）

### 5.2 认知纠偏师（增强版质询）
- [ ] 选中卡片群 → "启动辩论模式"
- [ ] AI 自动搜索卡片库中的反驳证据
- [ ] 生成"正方/反方"对比面板
- [ ] 纠偏结果可转为新卡片
- **预估**：3-4 天

### 5.3 研究路径简报
- [ ] 锁定一个主题卡片 → "生成研究路径"
- [ ] AI 分析关联卡片，生成研究路径图
- [ ] 列出必读材料、思考盲区、建议下一步
- [ ] 路径图可导出为 Markdown
- **预估**：3-4 天

### 5.4 搜索增强（语义搜索）
- [ ] 搜索框支持自然语言查询："所有支持方案A的论据"
- [ ] 后端基于 embedding 做语义检索
- [ ] 搜索结果高亮关联路径
- **预估**：4-5 天

### 5.5 主动通知机制
- [ ] 后台定期分析卡片库
- [ ] 发现潜在关联/矛盾时推送通知
- [ ] 通知可一键跳转到相关卡片
- **预估**：3-4 天

**Phase 5 总预估**：18-23 天

## Phase 6：捕捉层增强（Capture++）

> 目标：降低捕捉摩擦，支持多模态输入

### 6.1 截图捕捉
- [ ] 捕捉弹窗新增"截图"按钮
- [ ] 调用系统截图 API（Electron/Tauri 前期可用浏览器扩展过渡）
- [ ] 截图自动 OCR 提取文字
- [ ] 原图 + OCR 文字一起存为卡片
- **预估**：4-5 天

### 6.2 浏览器扩展
- [ ] Chrome/Firefox 扩展：划词发送到 Nexus
- [ ] 右键菜单"发送到 Nexus 画布"
- [ ] 自动抓取网页标题、URL、选中文本
- [ ] 扩展与后端 API 通信
- **预估**：5-6 天

### 6.3 语音速记
- [ ] 捕捉弹窗新增"语音"按钮
- [ ] 浏览器 MediaStream API 录音
- [ ] 调用 Whisper API 语音转文字
- [ ] 保留原音频 + 转写文字
- **预估**：3-4 天

### 6.4 智能预处理增强
- [ ] 捕捉时自动生成一句话摘要（已有标题压缩，扩展为摘要）
- [ ] 自动提取实体标签（人名、地名、概念）
- [ ] 自动判断卡片类型（观点/事实/问题/灵感）
- **预估**：2-3 天

**Phase 6 总预估**：14-18 天

## Phase 7：产出层构建（Produce）

> 目标：思考必须有出口，从"思维链"到"可交付物"

### 7.1 AI 文章生成
- [ ] 选中一组卡片 → "生成文章初稿"
- [ ] AI 基于思维链生成结构化文章
- [ ] 支持多种产出格式：文章/报告/方案/大纲
- [ ] 生成结果可编辑、可导出 Markdown
- **预估**：4-5 天

### 7.2 任务拆解
- [ ] 结论卡片 → "拆解为任务"
- [ ] AI 自动拆解为可执行的待办事项
- [ ] 任务带上下文背景（关联原始卡片）
- [ ] 任务可标记优先级、截止日期
- **预估**：3-4 天

### 7.3 导出系统
- [ ] 画布导出为 PNG/SVG 图片
- [ ] 卡片导出为 Markdown 文件
- [ ] 思维链导出为 JSON（可导入）
- [ ] 结论导出为 PDF 报告
- **预估**：3-4 天

### 7.4 与外部工具互通
- [ ] 导出到 Obsidian（Markdown 文件 + 双链）
- [ ] 导出到 Notion（API 集成）
- [ ] 任务同步到滴答清单/日历
- **预估**：5-7 天

**Phase 7 总预估**：15-20 天

- [ ] 搜索索引（本地全文索引）
- **预估**：3-4 天

### 8.3 UI/UX 细节
- [ ] 暗色/亮色主题切换
- [ ] 卡片模板系统（预设卡片类型）
- [ ] 画布迷你地图

- **预估**：3-4 天

**Phase 8 总预估**：10-13 天

## Phase 9：Tauri 桌面化

> 目标：获得系统级能力，实现"全域感知"

### 9.1 Tauri 迁移
- [ ] Rust 后端重写（SQLite、文件系统、加密）
- [ ] 现有 Web 前端直接嵌入 Tauri WebView
- [ ] Python LLM 客户端迁移到 Rust（reqwest + serde）
- **预估**：7-10 天

### 9.2 系统级能力
- [ ] 全局热键（任何应用中呼出 Nexus）
- [ ] 系统托盘 + 悬浮球
- [ ] 系统截图 API（无需浏览器扩展）
- [ ] 文件系统监听（自动导入指定目录的文件）
- **预估**：5-6 天

### 9.3 本地 AI 能力
- [ ] 本地 embedding 模型（语义搜索无需联网）
- [ ] 本地 Whisper 模型（语音转写无需联网）
- [ ] 隐私数据端到端加密
- **预估**：5-7 天

**Phase 9 总预估**：17-23 天

## 优先级排序

**立刻做（Phase 4）**：连线优化 → 分组 → 大纲视图
**核心价值（Phase 5）**：关系发现 → 语义搜索 → 辩论模式
**体验闭环（Phase 6-7）**：截图捕捉 → 文章生成 → 导出
**最终形态（Phase 9）**：Tauri 包装 → 全局热键 → 本地 AI
